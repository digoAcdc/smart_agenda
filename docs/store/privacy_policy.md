# Politica de Privacidade - Smart Agenda

Ultima atualizacao: 2026-03-03

## 1. Informacoes coletadas
O Smart Agenda foi desenhado para funcionar localmente no dispositivo. Nesta versao:

- Dados de agenda (eventos, grupos e grade horaria) ficam armazenados no aparelho.
- Imagens anexadas sao referenciadas por caminho local e salvas no armazenamento privado do app.
- Nao coletamos dados pessoais sensiveis para envio a servidores proprios.

## 2. Permissoes utilizadas

- `POST_NOTIFICATIONS`: para enviar lembretes locais de eventos.
- Acesso a midia/imagem (via seletor de imagem): para permitir anexos escolhidos pelo usuario.

## 3. Compartilhamento de dados
Nao compartilhamos dados de agenda com terceiros nesta versao.

## 4. Armazenamento e seguranca
Os dados ficam no banco local do app (Drift/SQLite). O usuario pode remover os dados desinstalando o app.

## 5. Direitos do usuario
O usuario pode editar e excluir seus eventos e grupos a qualquer momento no app.

## 6. Contato
Para duvidas sobre privacidade, entre em contato pelo email de suporte oficial do app.
