# Assets da Play Store - Checklist

## Graficos obrigatorios

- [ ] Icone 512x512 (PNG, fundo sem transparencia recomendada).
- [ ] Feature Graphic 1024x500.
- [ ] Capturas de tela reais:
  - [ ] Agenda (calendario + lista)
  - [ ] Busca com filtros
  - [ ] Grade horaria
  - [ ] Criacao/edicao de evento

## Textos de listagem

- [ ] Titulo: Smart Agenda
- [ ] Descricao curta (ate 80 chars)
- [ ] Descricao completa (beneficios + diferenciais + privacidade)
- [ ] Categoria e tags corretas
- [ ] Email/site de suporte

## Qualidade visual

- [ ] Mesma identidade de cores em todos os screenshots.
- [ ] Textos legiveis em tema claro e escuro.
- [ ] Sem dados pessoais nos screenshots.
